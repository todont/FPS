To run the project you need latest version of Unreal Engine 4(supported by versions later than 4.25.). 
You should use it to open the fps.project file. You can also use it to build a project.

Download link:
https://www.unrealengine.com/en-US/download?install=true

Link to the finished showcase project:
https://drive.google.com/drive/folders/1O0pHy75kYDpZ54Px6pJ2S5Aey5skIsiE?usp=sharing

Quick start:
1. Run the project
2. Open folder "Content"
3. Copy the folder "Content" or needed modules/blueprints to your project folder
4. Plug necessary modules to your character
